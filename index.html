Quadratic Equation Solver

Help

This program solves quadratic equations of the format: $$ax^2 + bx + c = 0$$ Then it solves it using the formula: $$x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$$ To get to this format use normal algebraic techniques. If you do not know how to do this, consult a book on the subject.
